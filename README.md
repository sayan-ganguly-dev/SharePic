# 📸 SharePic

A secure MERN-based image sharing platform where users can upload, browse, and manage images using JWT authentication and ImageKit cloud storage.

---

## 🚀 Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 🖼️ Upload Images using ImageKit
- 📰 Browse Image Feed
- ➕ Create New Posts
- ☁️ MongoDB Database
- ⚡ Fast React + Vite Frontend
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Vite
- CSS3
- Axios
- React Router DOM

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- ImageKit

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/sayan-ganguly-dev/SharePic-.git
```

## Backend

```bash
cd BACKEND
npm install
```

Create a `.env` file

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
IMAGEKIT_PRIVATE_KEY=your_private_key
IMAGEKIT_PUBLIC_KEY=your_public_key
IMAGEKIT_URL_ENDPOINT=your_url_endpoint
```

Run Backend

```bash
npm run dev
```

---

## Frontend

```bash
cd FRONTEND
npm install
npm run dev
```
